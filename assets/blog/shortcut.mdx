# 🚀 Shortcut VS Code yang Wajib Diketahui

Visual Studio Code (VS Code) adalah salah satu code editor paling populer dengan banyak shortcut yang bisa mempercepat workflow Anda. Berikut adalah beberapa shortcut penting yang wajib diketahui:

## ✨ Navigasi Cepat

- **Buka File Cepat**: `Ctrl + P`
- **Pindah Tab Editor**: `Ctrl + Tab`
- **Tutup Tab Saat Ini**: `Ctrl + W`
- **Buka Kembali Tab yang Ditutup**: `Ctrl + Shift + T`

## 📝 Editing

- **Pilih Semua**: `Ctrl + A`
- **Copy**: `Ctrl + C`, **Cut**: `Ctrl + X`, **Paste**: `Ctrl + V`
- **Undo**: `Ctrl + Z`, **Redo**: `Ctrl + Y`
- **Pindahkan Baris ke Atas/Bawah**: `Alt + ↑ / ↓`
- **Duplicate Baris**: `Shift + Alt + ↑ / ↓`
- **Multi Cursor (Tambah Cursor di Baris Lainnya)**: `Ctrl + Alt + ↑ / ↓`

## 🔍 Pencarian & Navigasi Kode

- **Cari dalam File**: `Ctrl + F`
- **Cari dan Ganti**: `Ctrl + H`
- **Cari di Semua File (Global Search)**: `Ctrl + Shift + F`
- **Lompat ke Baris Tertentu**: `Ctrl + G`

## ⚙️ Shortcut Lain yang Berguna

- **Toggle Sidebar**: `Ctrl + B`
- **Buka Terminal**: `Ctrl + ~`
- **Comment / Uncomment**: `Ctrl + /`
- **Format Kode Otomatis**: `Shift + Alt + F`

<Alert>
  💡 **Tip:** Anda bisa melihat dan mengatur shortcut di VS Code dengan menekan `Ctrl + K Ctrl + S`.
</Alert>

Semoga daftar ini membantu Anda bekerja lebih cepat dan efisien di VS Code! 🚀
